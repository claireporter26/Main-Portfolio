Frontend Mentor is a website which provides the user with ready-made designs and specifications for both mobile and desktop screens and provides the opportunity for peer feedback. The user then writes the code from scratch in order to create the design based on the specification.

This resource has allowed me to practice my coding without needing to spend too much time trying to create an exciting design.
